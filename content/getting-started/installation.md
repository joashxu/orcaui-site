This document install and setup your Django project.

### Django Slippers

You can install Slippers from PyPI

```bash
$ pip install slippers
```

Add `slippers` to your `INSTALLED_APPS`.

```python
# setting.py

INSTALLED_APPS = [
    ...,
    'slippers',
    ...,
]
```

And done.


## Tailwind CSS

You need Node.js and Tailwind CSS installed.
